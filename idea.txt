1. 데이터 기반 개선 전략
✅ 1-1. 학습 데이터 양 자체를 늘리기
효과: 특히 violence 클래스가 다양한 각도, 거리, 인원수, 배경에서 등장하도록 만들면, 모델이 일반화에 더 강해집니다.
실행 방법:
온라인에서 공개된 CCTV, 스포츠, 영화 영상 중 violence 포함된 클립 수집
현재 가지고 있는 영상에 frame interval 줄이기로 데이터 수 증가 (15fps → 5fps)
오픈소스 violence detection dataset 활용 (e.g., Surveillance Fight Dataset 등)

✅ 1-2. 비폭력(normal) 클래스 다양화
효과: violence와 헷갈릴 수 있는 동작들(격한 제스처, 스포츠, 무용 등)을 포함시켜 오탐률 감소
예시: 격렬한 동작이지만 비폭력인 영상 추가 (체조, 댄스, 군중 제스처 등)

2. 모델 구조 및 학습 전략 개선
✅ 2-1. LSTM → BiLSTM or Transformer 기반 구조로 전환
양방향 LSTM은 미세한 temporal 흐름을 더 잘 포착 가능
Temporal Transformer 기반 구조는 장기 시퀀스 관계 학습에 더 우수함

✅ 2-2. Multi-head Spatial Attention 추가
관절마다 중요한 조합을 모델이 더 세분화해서 학습 가능
현재는 joint별 단일 attention이지만, 복수 헤드 또는 관계 기반 GCN도 고려 가능

✅ 2-3. 이중 분류기 구조 (Cascade Classifier)
1차로 빠르고 민감한 탐지 → 2차로 정밀한 필터링 (예: LSTM 후 MLP 추가로 refinement)

3. 학습 기법 개선
✅ 3-1. Class Imbalance 처리
Focal Loss: 폭력 샘플을 더 민감하게 학습
Class Weights 적용: violence에 더 높은 손실 가중치 부여

✅ 3-2. Hard Example Mining
폭력 중에서도 오분류된 예제들만 모아서 재학습 → 가장 헷갈리는 케이스 집중 학습

✅ 3-3. Early stopping 제거 + 더 긴 학습
더 많은 epoch 학습이 필요한 구조라면 early stopping이 premature일 수 있음

4. 추론/알람 후처리 개선
✅ 4-1. Temporal Smoothing
한 시퀀스만 violence라고 바로 판단하지 않고, 연속된 다수 시퀀스가 일치할 때만 알람

✅ 4-2. Voting / Aggregation
한 인물에 대해 여러 시퀀스를 sliding window로 추론하고 voting 수행
